1. Что это такое?

Это пуржилка. Удаляет старые файлы из файлэх или перемещает их в архив.
Сама ищет директории внутри указанных корневых директорий и добавляет
их в список обрабатываемых. Удаляет из списка несуществующие директории.
Пишет в лог все, что делает.


2. Как это работает?

Настройки программы и комментарии к ним находятся в файле конфигурации.
При запуске без параметров показывается помощь. Краткое описание параметров:

-с  использовать альтернативный файл конфигурации.
-s  поиск новых директорий, добавление их в список, удаление несуществующих
    директорий из списка.
-p  пуржинг.
-q  не выводить ничего на экран.
