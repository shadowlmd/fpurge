Список изменений и нововведений.

Версия 0.1
* Первый релиз. Все тесты, которые пришли в голову, прошел успешно,
  глюков не обнаружено. :)

Версия 0.2
% Исправлен "идеологический" баг, который не влиял на работу программы.
* Для задания максимального возраста используется знак "|" вместо "?".

Версия 0.3
* При запуске без параметров теперь показывается помощь. Для пуржинга
  сделан отдельный параметр -p, который можно использовать одновременно
  с -s. В этом случае, сначала будем обновлять лист, потом пуржить.
* Часть кода немного переписана с целью оптимизации. :)
+ В ExcludeList'е теперь поддерживаются маски файлов.

Версия 0.31
* Для определения возраста файла используется время создания файла.
  Раньше использовалось время модификации.
* В дистрибутив добавлен скрипт FReport by Dmitry Mikchaylov (2:5066/26).

Версия 0.32
* Исходники версии 0.31 потерялись, и я не помню, как я в прошлый раз сделал
  получение времени создания файла, поэтому для определения возраста снова
  используется время модификации.
* Что-то прооптимизировал по мелочам. Возможно, что-то даже сломал. :)
% Самодельная функция для расчёта разницы в днях между двумя датами, которая
  не всегда работала точно, заменена на функцию из стандартной библиотеки,
  которая делает то же самое, но правильно.

Версия 0.33
* При перемещении файла в архив, добавляем его дату и время к имени.
+ Добавлен параметр конфига UseCreationTime - использовать время создания
  файла для определения возраста. Полезно, если по файлэхе прилетает какая-то
  древность, а файлэхопроцессор не умеет ставить текущую дату на вновь
  пришедший файл. Пока что фича работает только под Windows.
